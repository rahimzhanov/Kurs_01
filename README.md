# Проект "Анализ банковских операций"

Проект предназначен для анализа транзакций по банковским картам и генерации финансовых отчётов на основе Excel-файла с данными.

## Функционал

- Генерация приветствия в зависимости от времени суток.
- Отображение информации по картам (общие траты, кэшбэк).
- Топ-5 самых крупных транзакций.
- Получение курсов валют (через ЦБ РФ).
- Получение цен на акции (через Alpha Vantage).
- Поиск транзакций по описанию или категории.
- Расчёт средних трат по дням недели за последние 3 месяца.

## Зависимости

- Python 3.9+
- pandas
- requests
- openpyxl
- python-dotenv

Установка зависимостей:
```bash
pip install pandas requests openpyxl python-dotenv
